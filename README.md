# Eyal Segev — Portfolio

Personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Stack

- React 18 + Vite 5
- Tailwind CSS v3
- Framer Motion (scroll animations + 3D effects)
- Lenis (smooth scroll)
- EmailJS (contact form)

## Interactive Effects

- Aurora animated background — 3 drifting CSS blobs in the hero
- Spotlight — cursor-tracked radial glow across the hero section
- 3D card tilt — project cards tilt ±10° with holographic glare on hover
- Magnetic buttons — CTAs spring-follow the cursor within 80px

## Running locally

```bash
cd frontend
npm install
cp .env.example .env   # fill in your EmailJS credentials
npm start
```

Opens at `http://localhost:5173`.
